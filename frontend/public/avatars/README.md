# Custom Avatar Images

Place your custom avatar images in this directory.

## File Naming Convention
- Use user email or ID as filename
- Supported formats: .png, .jpg, .svg
- Example: `maya@example.com.png`

## Recommended Specifications
- Size: 512x512px or larger
- Format: PNG with transparency preferred
- Style: Match the futuristic Web3 aesthetic shown in reference images

## Usage
The system will automatically use these custom avatars when available,
falling back to generated avatars if a custom image is not found.
